## Project Title
Rule-Based AI Chatbot

## Description
This project is a simple chatbot built using Python. It uses if-else logic and keyword matching to respond to user questions.

## Features
- Responds to greetings like "hello"
- Explains what AI is
- Answers questions about the course
- Provides help to users
- Gives polite responses

## Technologies Used
- Python

## How to Run the Project
1. Install Python on your computer
2. Open the project folder
3. Run the file:
   python chatbot.py

## How It Works
The chatbot uses a while loop to continuously accept user input. It checks for keywords in the input using if-else statements and returns appropriate responses.

## Example Inputs
- hello
- what is AI
- help
